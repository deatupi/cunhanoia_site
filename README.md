# Cunha Noia Advocacia

Este é o repositório do site oficial da **Cunha Noia Advocacia**, especializada em Direito de Família e Direito da Saúde no Rio de Janeiro.

## 🚀 Como subir no Cloudflare Pages

1. Crie um novo projeto no [Cloudflare Pages](https://pages.cloudflare.com/).
2. Conecte seu repositório GitHub.
3. Configure o diretório de build como vazio (raiz) e o comando de build como vazio.
4. O site será publicado automaticamente no domínio `.pages.dev`.

## 📂 Estrutura do Projeto

- `index.html`: Página principal com estilos (CSS) e scripts (JS) inline para máxima performance.
- `images/`: Pasta contendo o logotipo e imagens do Dr. Luciano Cunha Noia.
- `_headers`: Configurações de segurança e cache para o Cloudflare.
- `_redirects`: Configurações de redirecionamento (se necessário).

## 🛠️ Tecnologias
- HTML5 / CSS3 (Mobile First)
- JavaScript Vanilla
- SEO Avançado (JSON-LD, Open Graph)
